# Wordle

## Running the App
1. Clone the repository:

   ```
   git clone <repository-url>
   cd <repository-name>
   ```

2. Copy .env.example to .env and fill in the necessary environment variables:
    ```
    cp .env.sample .env
    ```
3. Start Service by:
    ```sh
    docker compose up -d
    ```
    then open http://localhost:8711
